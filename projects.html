Форматы сведений об информационных ресурсах

А Б В Г Д И К М О П Р С Ф X A B D E F H I J M O R S

А

Аннотация Annotation
Краткое описание содержания ресурса с указанием основных характеристик и особенностей.
Формат: текстовый документ, 200-500 символов. Используется в библиографических описаниях.
Атрибутивный поиск Attribute search
Поиск информации по заданным атрибутам или характеристикам документа.
Применяется в базах данных, каталогах. Пример: поиск по автору, дате, ключевым словам.

Б

Библиографическая запись Bibliographic record
Структурированная информация о документе, включающая основные идентификаторы и метаданные.
Стандарт: ГОСТ 7.1-2003. Форматы: MARC21, UNIMARC, RUSMARC.
Базис классификации Classification basis
Совокупность признаков, по которым осуществляется разделение объектов классификации.
Примеры: тематический, хронологический, географический базисы.

В

Вторичные документы Secondary documents
Документы, созданные в результате аналитико-синтетической переработки первичных документов.
Примеры: рефераты, обзоры, аннотации, библиографические указатели.

Г

Гипертекстовая разметка Hypertext markup
Способ организации документов с использованием гиперссылок между различными фрагментами текста.
Стандарты: HTML, XML. Применяется в веб-документах, электронных изданиях.

Д

Дублинское ядро Dublin Core
Международный стандарт метаданных для описания цифровых ресурсов, включающий 15 основных элементов.
Элементы: Title, Creator, Subject, Description и др. Применяется в цифровых библиотеках.
Дескриптор Descriptor
Лексическая единица информационно-поискового языка, используемая для описания содержания документа.
Применяется в информационно-поисковых системах, тезаурусах.

И

Информационный массив Information array
Упорядоченная совокупность документов или данных, объединенных по определенным признакам.
Примеры: базы данных, архивы, коллекции цифровых объектов.
Идентификатор объекта Object identifier
Уникальный признак, позволяющий однозначно определить информационный объект в системе.
Форматы: DOI, ISBN, ISSN, URI. Стандарты: ISO 26324, ISO 2108.

К

Кластерный анализ Cluster analysis
Метод автоматической группировки документов по степени сходства их содержания.
Применяется в информационном поиске, анализе текстов, категоризации.
Контролируемый словарь Controlled vocabulary
Ограниченный набор терминов, используемый для индексирования и поиска информации.
Примеры: тезаурусы, рубрикаторы, классификационные схемы.

М

Метаданные Metadata
Структурированная информация, описывающая характеристики и содержание информационного ресурса.
Типы: описательные, административные, структурные. Стандарты: ISO 15836, ГОСТ Р 7.0.3-2006.
Многоаспектная классификация Multidimensional classification
Система классификации, учитывающая несколько независимых признаков объектов.
Применяется в библиотечных классификациях, электронных каталогах.

О

Онтология Ontology
Формальное описание системы понятий и отношений между ними в определенной предметной области.
Используется в семантическом вебе, искусственном интеллекте. Форматы: OWL, RDF.

П

Профиль метаданных Metadata profile
Набор элементов метаданных, адаптированный для конкретной области применения.
Примеры: профили для описания образовательных ресурсов, музейных объектов.

Р

Реферат Abstract
Сжатое изложение содержания документа с основными фактическими сведениями и выводами.
Стандарты: ГОСТ 7.9-95. Объем: 500-1000 знаков для научных публикаций.

С

Семантическая разметка Semantic markup
Способ описания содержания документа с указанием смысла его элементов.
Технологии: RDF, микроформаты, schema.org. Применяется в семантическом вебе.

Ф

Фасетная классификация Faceted classification
Система организации информационных ресурсов по независимым аспектам (фасетам).
Применение: электронные каталоги, интернет-магазины. Пример: классификация товаров по цвету, размеру, бренду.
Формальная онтология Formal ontology
Точная спецификация концептуальной модели предметной области с использованием формальных языков.
Используется в компьютерных науках, искусственном интеллекте. Языки: OWL, KIF.

X

XML-схема XML Schema
Способ описания структуры и содержания XML-документов, используемый для валидации метаданных.
Применяется в форматах MODS, METS. Стандарт: W3C XML Schema.

A

API Application Programming Interface
Набор протоколов и инструментов для взаимодействия между программными компонентами.
Используется для доступа к метаданным и контенту информационных систем.

B

BIBFRAME Bibliographic Framework
Модель описания библиографических данных, основанная на технологиях семантического веба.
Разработана Библиотекой Конгресса как замена формата MARC.

D

DOI Digital Object Identifier
Уникальный постоянный идентификатор цифрового объекта в сети Интернет.
Формат: 10.XXXX/XXXXX. Применяется для научных публикаций, данных.
Dublin Core Дублинское ядро
Международный стандарт метаданных для описания цифровых ресурсов.
Включает 15 основных элементов. Стандарт: ISO 15836.

E

EAD Encoded Archival Description
Стандарт кодирования архивных описаний на основе XML.
Используется для описания архивных коллекций, рукописей.

F

FRBR Functional Requirements for Bibliographic Records
Концептуальная модель библиографической записи, разработанная IFLA.
Определяет четыре уровня сущностей: Work, Expression, Manifestation, Item.

H

HTML5 Microdata Микроданные HTML5
Спецификация для встраивания семантической разметки в веб-страницы.
Используется совместно со словарями schema.org для обогащения контента.

I

IPTC International Press Telecommunications Council
Стандарт метаданных для описания фотографий и мультимедийного контента.
Широко используется в СМИ, фотобанках. Включает заголовки, ключевые слова, авторство.

J

JSON-LD JSON for Linked Data
Формат представления связанных данных на основе JSON.
Используется для семантической разметки веб-страниц, API. Стандарт W3C.

M

METS Metadata Encoding and Transmission Standard
Стандарт для кодирования описательных, административных и структурных метаданных.
Используется в цифровых библиотеках. Основан на XML.
MODS Metadata Object Description Schema
Схема метаданных для библиографического описания, альтернатива MARC.
Разработана Библиотекой Конгресса. Основана на XML.

O

OAI-PMH Open Archives Initiative Protocol for Metadata Harvesting
Протокол сбора метаданных из распределенных репозиториев.
Используется в цифровых библиотеках, научных архивах.

R

RDF Resource Description Framework
Фреймворк для описания ресурсов в виде триплетов "субъект-предикат-объект".
Основа семантического веба. Стандарт W3C.

S

SKOS Simple Knowledge Organization System
Модель представления систем организации знаний (тезаурусов, классификаций).
Основана на RDF. Используется для публикации словарей в семантическом вебе.
